# Chrome Web Store Listing

## Name
Resume Studio - AI Resume Generator from Job Descriptions

## Short Description (132 chars max)
Generate ATS-optimized resumes from any job posting. One click on Indeed, LinkedIn, Glassdoor, and more.

## Detailed Description
Resume Studio captures job descriptions directly from job boards and generates a complete, tailored application package in seconds.

**How it works:**
1. Browse jobs on Indeed, LinkedIn, Glassdoor, Monster, Dice, or ZipRecruiter
2. Click the "Generate Resume" button that appears on job listings
3. Resume Studio extracts the job description and creates:
   - ATS-optimized resume matched to the role
   - Personalized cover letter
   - LinkedIn summary
   - Cold outreach email
   - Interview preparation guide
   - Certification recommendations

**Supported Job Boards:**
- Indeed
- LinkedIn Jobs
- Glassdoor
- Monster
- Dice
- ZipRecruiter

**Key Features:**
- One-click resume generation from any supported job board
- AI-powered ATS keyword optimization
- 6 document types generated simultaneously
- Smart job description parsing
- Secure authentication with your Resume Studio account

**Privacy:**
- We only access job description content on supported job boards
- No browsing data is collected or transmitted
- Authentication tokens are stored locally in your browser
- All data processing happens on Resume Studio servers with encryption

**Requirements:**
- Free Resume Studio account (2 free documents/month)
- Pro subscription for unlimited generation
